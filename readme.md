# <p align="center">LEDDs v0.1</p>

<p align="center">
  Wi-Fi connected dimmer for LED bulbs.
</p>

> Live voltage!

## Characteristics
*Can be used to replace a bulb switch.*

 - trailing-edge
 - rotary encoder for physical control
